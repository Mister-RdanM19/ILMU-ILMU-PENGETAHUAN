## 📌 Deskripsi Proyek
Proyek ini adalah perpustakaan digital mandiri yang mengintegrasikan berbagai disiplin ilmu pengetahuan ke dalam satu sistem interface web yang responsif dan profesional. Dirancang dengan estetika industrial yang kuat, sistem ini berfungsi sebagai pusat penyimpanan data edukasi yang terorganisir.

## 🛠 Spesifikasi Teknis
Sistem ini dibangun menggunakan arsitektur web modern yang ringan namun fungsional:
* **Core Engine:** HTML5, CSS3 (Modern Flexbox & Grid System)
* **Typography:** Oswald (Racing Speed Style) & Inter UI
* **Iconography:** FontAwesome 6.5.1
* **UI/UX:** Industrial Technical Theme (Dark Mode Optimized)
